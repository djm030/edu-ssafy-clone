# OMX Sharing Policy

This repository tracks only stable OMX artifacts that help collaborators understand and reproduce the planning harness.

Tracked:
- `.omx/context/**`
- `.omx/plans/**`
- `.omx/wiki/**`
- `.omx/notepad.md`
- `.omx/project-memory.json`

Not tracked:
- `.omx/logs/**`
- `.omx/state/**`
- `.omx/metrics.json`
- `.omx/hud-config.json`
- `.omx/setup-scope.json`

Do not commit runtime session state, hook logs, local machine paths, or agent session history.
